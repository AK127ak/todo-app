# To-Do List с категориями

Веб-приложение для управления задачами с возможностью организации по категориям.

## Функциональность

-  Создание, редактирование и удаление задач
-  Организация задач по категориям
-  Установка приоритетов и сроков выполнения
-  Аутентификация и авторизация пользователей
-  Адаптивный дизайн
-  Поиск и фильтрация задач
